MonoSid - AI-Powered Therapy App (Prototype)

MonoSid is an AI-powered virtual therapist designed to help people with mental health issues, focusing on depression, anxiety and trauma. This prototype version aims to provide a basic text-based interaction with Sky, the virtual therapist, using GPT-3.5.

Prototype Features
- Text-based conversation: Chat with Sky (your virtual therapist) through text-based messages.
- Personalized Responses: Sky will respond to user inputs using GPT-3.5, tailored to the user’s needs.
- Conversational Memory: The prototype will remember past conversations within the same session.
- Basic User Interface: A simple chat interface for sending and receiving messages.

 Goal of the Prototype
The goal is to create a working text-based chatbot that provides therapeutic conversations. This prototype will test the concept of personalized AI therapy and help identify areas for improvement before adding more advanced features.
